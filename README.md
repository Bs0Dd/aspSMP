# aspSMP - программатор для СМП МК90 на основе USBasp

![aspSMP](https://raw.githubusercontent.com/Bs0Dd/aspSMP/master/graphics/aspSMP.jpg)

Прошивка и ПО для популярного и дешевого программатора USBasp, превращающая его в программатор
Сменных Модулей Памяти (СМП) для микрокомпьютера Электроника МК90 (модули МП-100-0, СМП-0, МПО-10).

Основано на проекте SMPReaderUSB - https://mk90.blogspot.com/2009/11/smpreaderusb.html

Инструкция по использованию: [MANUAL.md](https://github.com/Bs0Dd/aspSMP/blob/master/MANUAL.md)
